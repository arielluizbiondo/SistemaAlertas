# 🚨 Sistema de Alertas com Simulação de Sensores

Este projeto é um **sistema simples de alertas**, desenvolvido utilizando **PHP, HTML e CSS**.

O sistema permite **simular sensores de diferentes setores**, registrando alertas que podem ser visualizados posteriormente.

---

# ⚙️ Funcionalidades

✔ Simulação de sensores em diferentes setores
✔ Registro automático de alertas com data e hora
✔ Visualização de todos os alertas registrados
✔ Remoção de alertas resolvidos

### Simular sensores

Arquivo: **simular.php**

Nesta página é possível simular sensores em diferentes setores:

* Sala
* Garagem
* Quarto
* Cozinha

Ao clicar em um botão, o sistema registra automaticamente um **alerta com o nome do setor e a data/hora atual**.
